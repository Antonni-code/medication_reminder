Smart Medication Reminder Box - Wiring Diagram

Complete System with Power Control & Menu Navigation

5 NAVIGATION BUTTONS + ON/OFF POWER SWITCH
Red = 5V Power
Black = Ground (GND)
Blue = SDA (Data)
Purple = SCL (Clock)
Orange = Digital Signals
Pink = Power Switch Line

OLED Display (SCL) A5 I2C Clock - shared with RTC

LED Thursday D5 220Ω resistor required

Buzzer D9 Active buzzer, 5V
UP Button D10 10kΩ pull-up to 5V
DOWN Button D11 10kΩ pull-up to 5V
SET Button D12 10kΩ pull-up to 5V
CONFIRM Button D13 10kΩ pull-up to 5V
HOME Button (NEW) A0 10kΩ pull-up to 5V - Exit menu/return home
ON/OFF Switch (NEW) N/A Hardware switch - in power line before Arduino

Button Functions:

  • ON/OFF Switch: Hardware toggle - physically cuts power to entire system
  • HOME Button (Pin A0): Exit menu mode without saving / Return to main display
  • UP Button (Pin 10): Navigate up in menu / Increase hour or minute
  • DOWN Button (Pin 11): Navigate down in menu / Decrease hour or minute
  • SET Button (Pin 12): Enter menu / Save settings / Move to next field
  • CONFIRM Button (Pin 13): Confirm medication taken / Acknowledge alarm

Important Notes:

  • ON/OFF Switch: SPST toggle switch in the 5V line BEFORE Arduino - no pin needed
  • HOME Button: Uses analog pin A0 as digital input - same wiring as other buttons
  • 5 Buttons Total: UP, DOWN, SET, CONFIRM (pins 10-13) + HOME (pin A0)
  • Button Wiring: One leg to Arduino pin, other leg to GND, 10kΩ resistor to 5V
  • I2C Devices: OLED and RTC share SDA (A4) and SCL (A5) - this is normal
  • LED Direction: Long leg (anode) to Arduino pin through 220Ω resistor, short leg to GND
  • Default Times: Morning 8:00 AM, Afternoon 1:00 PM, Evening 8:00 PM
